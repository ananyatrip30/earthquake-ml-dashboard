# 🌍 Earthquake ML Dashboard

An **AI-powered earthquake prediction dashboard** that uses Machine Learning and real-time data to analyze and predict earthquake magnitudes.

---

## 🚀 Features

- 🌍 **Live Earthquake Data** using USGS API  
- 🗺️ **Interactive Map Visualization** (multiple earthquakes)  
- 🤖 **Machine Learning Prediction** (Random Forest model)  
- 🚨 **Auto Alert System** for high magnitude earthquakes  
- 📊 **Real Dataset Analysis & Graphs**  
- 🔄 **Auto Refresh Dashboard**  
- 🎨 Clean & user-friendly UI built with Streamlit  

---

## 🧠 Machine Learning

- Model Used: **Random Forest Regressor**
- Target: Earthquake Magnitude Prediction
- Evaluation Metric: **Mean Absolute Error (MAE)**

---

## 📊 Dataset

- Source: Earthquake dataset (1995–2023)
- Features include:
  - Latitude, Longitude  
  - Depth  
  - Magnitude indicators (CDI, MMI, etc.)  
  - Date features (Year, Month, Day)  

---

## 🛠️ Tech Stack

- Python 🐍  
- Pandas & NumPy  
- Scikit-learn  
- Matplotlib  
- Streamlit  
- USGS Earthquake API
